# CodeGen

Simple and succint code generation for your libraries.

## Installation

When [available in Hex](https://hex.pm/docs/publish), the package can be installed
by adding `code_gen` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:code_gen, "~> 0.1.0"}
  ]
end
```
